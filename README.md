  <h1 style="border-bottom: 2px solid #eaecef; padding-bottom: 0.3em;">
    Nexus: Investor & Entrepreneur Collaboration Platform
  </h1>
  
  <p>
    A full-stack web application designed to bridge the gap between startup founders and venture capitalists. Nexus provides a secure ecosystem for discovering opportunities, scheduling meetings, conducting video calls, and managing investment documents.
  </p>

  <h2>🚀 Tech Stack</h2>
  <ul>
    <li><strong>Frontend:</strong> React 18, TypeScript, Tailwind CSS, Vite</li>
    <li><strong>Backend:</strong> Node.js, Express.js</li>
    <li><strong>Database:</strong> MongoDB (with Mongoose for Polymorphic Schemas)</li>
    <li><strong>Real-time & Security:</strong> WebRTC, Socket.IO, JWT Auth, bcrypt</li>
  </ul>

  <h2>✨ Key Features</h2>
  <ul>
    <li><strong>Secure Authentication:</strong> JWT-based login with role-based access control (RBAC) for Investors and Entrepreneurs.</li>
    <li><strong>Polymorphic Profiles:</strong> Dynamic data structures allowing unique profile fields depending on the user's role.</li>
    <li><strong>Collaboration Engine:</strong> Request system for initiating investment conversations.</li>
    <li><strong>Virtual Chamber:</strong> Integrated meeting scheduling and WebRTC video calling.</li>
    <li><strong>Document Processing:</strong> Secure upload and management of pitch decks and financial files.</li>
  </ul>
